Programs and Why to Use Them?

Which one do I use?

In Animation and Design, The are 2 Main categories: 3D and 2D. Choosing which style and software can make a big difference. Some Programs have the ability to do real-life lighting while others are for more of a blocky style. A program can make a difference in art style to! Let's see the differences as well as the Pros and Cons of each Program!

Block Bench

Avantage

  • Easy for beginner
  • Can run on any device
  • Simple and Easy to learn UI

Disavantage

  • Can not do Lighting
  • Limited to Voxels

Blender

Blender is a Program that is designed for more complicated scenes. It adds many extra features which can help an Animation look more realistic

Advantage

  • Easy for beginner
  • Much better lighting
  • Bigger Community

Disadvantage

  • Requires a Powerful PC
  • Takes time to Render an Image/Animation
